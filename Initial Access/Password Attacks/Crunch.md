# crunch

`@`  - caractères alphabétiques minuscules
`, ` - caractères alphabétiques majuscules
`%`  - caractères numériques
`^ ` - caractères spéciaux incluant l'espace


```sh
crunch 8 8 -t ,@@^^%%%
```

```sh
crunch 4 6 -f /usr/share/crunch/charset.lst mixalpha -o output.txt
```
